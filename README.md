# 🌙 Luna: AI Chat & Code Quality Analyzer

An intelligent, multi-provider code quality analysis and automated refactoring tool designed to assist software developers directly inside their active development workspace.

---

## 🚀 Key Features

- **Multi-Cloud Architecture:** Smoothly toggle between state-of-the-art AI engines like **Groq, Gemini, and Anthropic** depending on your engineering task.
- **Centralized Communication Layer:** Decoupled architecture that manages all external API integrations, secure authentication tokens, and response parsing.
- **Automated Payload Formatting:** Instantly maps generic user chat prompts into specialized JSON schemas required by the active cloud provider.
- **Native IDE Workspace Panel:** Interactive React Webview dashboard running seamlessly inside a TypeScript-powered VS Code extension framework.

---

## 🛠️ Tech Stack & Architecture

- **Core Backend Engine:** Node.js, TypeScript, VS Code Extension API (`vsce`)
- **Interactive UI Dashboard:** React.js, Webview HTML Bridges
- **Supported AI Integrations:** Groq API, Google Gemini, Anthropic Claude
